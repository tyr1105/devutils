# DevUtils - Free Online Developer Tools

> 8 essential developer tools in one beautiful, dark-themed interface. Free, no signup, works offline.

**Live Demo:** [https://tyr1105.github.io/devutils](https://tyr1105.github.io/devutils)

## Tools Included

| Tool | Description |
|------|-------------|
| **JSON Formatter** | Format, minify, and validate JSON |
| **Base64** | Encode and decode Base64 strings |
| **JWT Decoder** | Decode and inspect JSON Web Tokens |
| **Regex Tester** | Test regex with real-time matching & groups |
| **UUID Generator** | Generate single or batch UUIDs (v4) |
| **URL Encoder** | Encode/decode URL components |
| **HTML Entities** | Encode/decode HTML entities |
| **Hash Generator** | SHA-1, SHA-256, SHA-512 hashes |

## Features

- Dark theme, clean UI
- Client-side only — no data sent to any server
- Works offline (cache it once, use anywhere)
- Mobile responsive
- Copy to clipboard with one click

## Tech

- Vanilla HTML/CSS/JS — zero dependencies
- Web Crypto API for hashing
- `crypto.randomUUID()` for UUID generation
- Hosted on GitHub Pages (free)
