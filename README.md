# ⚡ DevUtils

**Free, fast, privacy-first developer tools.** 12 essential tools that work entirely in your browser.

## 🔧 Tools

| Tool | Description |
|------|-------------|
| [JSON Formatter](https://tyr1105.github.io/devutils/tools/json-formatter.html) | Format, validate and beautify JSON |
| [Base64 Codec](https://tyr1105.github.io/devutils/tools/base64.html) | Encode and decode Base64 strings |
| [JWT Decoder](https://tyr1105.github.io/devutils/tools/jwt-decoder.html) | Decode and inspect JSON Web Tokens |
| [Regex Tester](https://tyr1105.github.io/devutils/tools/regex-tester.html) | Test regular expressions with live matching |
| [Color Picker](https://tyr1105.github.io/devutils/tools/color-picker.html) | Convert colors between HEX, RGB, HSL |
| [CSS Gradient](https://tyr1105.github.io/devutils/tools/css-gradient.html) | Generate beautiful CSS gradients |
| [Markdown Preview](https://tyr1105.github.io/devutils/tools/markdown.html) | Write and preview Markdown in real-time |
| [QR Generator](https://tyr1105.github.io/devutils/tools/qr-generator.html) | Generate QR codes from any text or URL |
| [URL Codec](https://tyr1105.github.io/devutils/tools/url-codec.html) | Encode and decode URLs |
| [Hash Generator](https://tyr1105.github.io/devutils/tools/hash-generator.html) | Generate MD5, SHA-1, SHA-256, SHA-512 |
| [Timestamp](https://tyr1105.github.io/devutils/tools/timestamp.html) | Convert Unix timestamps to dates |
| [Lorem Ipsum](https://tyr1105.github.io/devutils/tools/lorem-ipsum.html) | Generate placeholder text |

## ✨ Features

- 🔒 **Privacy First** — All processing happens in your browser, no data sent to servers
- ⚡ **Instant Results** — Everything works offline, no waiting
- 📱 **Works Everywhere** — Responsive design, works on mobile
- 🆓 **Always Free** — No sign-up, no ads, no tracking
